# OG Images Directory

This directory should contain Open Graph images for each page and locale.

## Required Images

Create the following images (1200x630px, JPEG format):

- `home-en.jpg` - Home page (English)
- `home-ar.jpg` - Home page (Arabic)
- `about-en.jpg` - About page (English)
- `about-ar.jpg` - About page (Arabic)
- `services-en.jpg` - Services page (English)
- `services-ar.jpg` - Services page (Arabic)
- `work-en.jpg` - Work page (English)
- `work-ar.jpg` - Work page (Arabic)
- `contact-en.jpg` - Contact page (English)
- `contact-ar.jpg` - Contact page (Arabic)

## Design Guidelines

- Size: 1200x630px (1.91:1 ratio)
- Format: JPEG (optimized)
- Include: Brand logo, page title, key visual elements
- Text: Should be readable at small sizes (social media previews)
- Colors: Use brand colors (Teal/Navy from constants.ts)

## Quick Generation

You can use tools like:
- Canva (OG Image templates)
- Figma (1200x630 artboard)
- Online OG image generators
